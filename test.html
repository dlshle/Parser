Welcome to try out my shabby XText parser.

Here's some syntax for this shabby parser:

Hyper link:

[a:link]text or something else[/a]

for example:

[a:www.google.com]Go to google[/a]

or

[a:www.google.com][img:logo.png]image of google[/img][/a]

real shabby huh?

Image:

[img:image_src.png h:100 w:100]description[/img]

h and w are height and weight attribution of this img tag. You can just specify h or w, it's okay.

for example:

[img:a_pic.jpg]a picture[/img]

[img:b_pic.png h:123]meh[/img]

I know you are thinking "Isn't it just a different way to write HTML?" Yeah, you are right!

Code:

[code:language]

code...

[/code]

language is the programming language you are gonna write or paste.

for example:

[code:c]
#include

int main()
{
	printf("Hello world!");
	return 0;
}
[/code]
